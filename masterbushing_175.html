17.5 Rotary Table Master Bushing Catalogue  

Keeping in view requirement of shallow to medium depth wells SDTI-MB - 27550 PSP have been developed by Spectrum Drilling Tools International as per API-7K specifications suitable for 27-½" and sizes of Rotary Tables. Pin drive has same advantages as in SDTI-MB 2750 PS. The SDTI-MB 27550 PSP master bushings are provided with an extendedAPI taper in the bowls. Rise in taper by 2-½" inches, makes all slips rise by 2-½" which makes slip handling easier. This rise of slips distributes the load uniformly in the bushing resulting in less wear & tear of table bore. Additionally this taper provides 50% more slip support resulting in better load managing capability



SOLID BODY SQUARE DRIVE MASTER BUSHING




SDTI-27550 SS is used for all drilling operations up to 10000 feet. Square Drive Master Bushing SDTI is used where a large slip back-up area is not needed. With the help of different sizes extended API insert bowls SDTI can handle tubular of 2-3/8" to 13-3/8" OD and SDTI - 20550 SS can handle tabular sizes of The solid outer body is designed in a way that it supports all transverse load and imparts suitable backing for split insert bowls even when the rotary table is in use for drilling operation.



SPLIT TYPE PIN DRIVE MASTER BUSHING




Spectrum Drilling Tools International has developed special type Split Type Pin Drive Master Bushing suitable for 27½” API Rotary Table as per API-7K specifications which can handle tubular size from 2-3/8” to 13-3/8” OD with the help of bowl no.1,2 &3 and MasterBushing designed to handle string load of 750 ton.
Main advantages of this special Split Type Pin Drive Master Bushing is as under:-
  • Split Type Pin Drive Master Bush can be remove from the drilling string to pass large diameter bit and pipe connection directly through the rotary table
  • BOP can be easily lifted by removing the Split Type Master Bush from the Rotary Table
  • Master Bush can be lock with the Rotary Table and Master Bush has locks that hold the bowls securely in the bushing.
  • String and near bit stabilizers can be used with removing Split Type Master Bushing from the Rotary Table
