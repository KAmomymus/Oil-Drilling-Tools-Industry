Solid Body Pin Drive Master Bushing Catalogue  

Spectrum Drilling Tools International has developed Solid Body Pin Drive master bushing as per API - 7K specifications taking into account different sizes of rotary tables i.e. 27-½. The advantage of using Pin Drive Kelly Bushings makes it possible to use extended split bowls in the rotary table while the Kelly bushing is in operation.The extended bowls are made in three sizes as perAPI for different sizes of tubulars as given below :- Extended bowls provide better slip back ups & this back up is made use in running in heavier tubular strings with out string failure due to bottle necking. SDTI 27550 PS with appropriate insert bowl in use can hold maximum of 750 tonsstring load.



SOLID BODY SQUARE DRIVE MASTER BUSHING




17-½" Through 27-½" SDTI-27550 SS is used for all drilling operations up to 10000 feet. Square Drive Master Bushing SDTI is used where a large slip back-up area is not needed. With the help of different sizes extended API insert bowls SDTI can handle tubular of 2-3/8" to 13-3/8" OD and SDTI - 20550 SS can handle tabular sizes of The solid outer body is designed in a way that it supports all transverse load and imparts suitable backing for split insert bowls even when the rotary table is in use for drilling operation.



SPLIT TYPE PIN DRIVE MASTER BUSHING




Spectrum Drilling Tools International has developed special type Split Type Pin Drive Master Bushing suitable for 27½” API Rotary Table as per API-7K specifications which can handle tubular size from 2-3/8” to 13-3/8” OD with the help of bowl no.1,2 &3 and MasterBushing designed to handle string load of 750 ton.
Main advantages of this special Split Type Pin Drive Master Bushing is as under:-
  • Split Type Pin Drive Master Bush can be remove from the drilling string to pass large diameter bit and pipe connection directly through the rotary table
  • BOP can be easily lifted by removing the Split Type Master Bush from the Rotary Table
  • Master Bush can be lock with the Rotary Table and Master Bush has locks that hold the bowls securely in the bushing.
  • String and near bit stabilizers can be used with removing Split Type Master Bushing from the Rotary Table
