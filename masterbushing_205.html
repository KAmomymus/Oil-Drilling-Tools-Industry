SOLID BODY PIN DRIVE MASTER BUSHING

For National, EMSCO, OilWell Rotary Table



Spectrum Drilling Tools International has developed Solid Body Pin Drive master bushing as per API - 7K specifications taking into account sizes of rotary tables i.e. 20-½. The advantage of using Pin Drive Kelly Bushings makes it possible to use extended split bowls in the rotary table while the Kelly bushing is in operation. The extended bowls are made in two sizes as per API for different sizes of tubulars.

Extended bowls provide better slip back ups & this back up is made use in running in heavier tubular strings with out string failure due to bottle necking. SDTI 20550 PS with appropriate insert bowl in use can hold maximum of 500 ton string load.

Following Bowls available for 20.5” Solid Body Master Bushing.
  • Insert Bowl No. 2 to handle tubular size 9.5/8” to 10.3/4” tubulars.
  • Insert Bowl No. 3 to handle tubular size 2.3/8” to 8.5/8” tubulars.



MEDIUM DUTY PIN DRIVE SPLIT MASTER BUSHING

For National, EMSCO, OilWell Rotary Table



Keeping in view requirement of shallow to medium depth wells SDTI-MB - 20550 PSP have been developed by Spectrum Drilling Tools International as per API-7K specifications suitable for 20-½" Rotary Tables. The SDTI-MB 20550 PSP Master Bushings are provided with an extended API taper in the bowls. Rise in taper by 2-½" inches, makes all slips rise by 2-½" which makes slip handling easier. This rise of slips distributes the load uniformly in the bushing resulting in less wear & tear of table bore. Additionally, this taper provides 50% more slip support resulting in better load managing capability.
